# AI-EcoGrow — Smart IoT Incubator MVP

Өндірістік практика аясында әзірленген, AI қолдауы бар ақылды жылыжай / зауыт микроклиматын басқару жүйесі.

## 🚀 Функционалы:
- **IoT Dashboard:** Нақты уақыттағы температура мен ылғалдылық графиктері.
- **Automation Module:** Температура критикалық деңгейге жеткенде автоматты желдеткіш пен помпаны іске қосу симуляциясы.
- **AI Analytics:** Деректерді талдап, агрономиялық есеп беретін жасанды интеллект модулі.

## 🛠️ Технологиялық стек:
- Python 3.10+
- Streamlit (UI & Landing page)
- Plotly (Деректерді визуализациялау)
